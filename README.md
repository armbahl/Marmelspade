# Marmelspade - Creator Jam Edition
---
This version is used for populating the Creator Jam search. It does <u>not</u> work with your own/group private inventory, only publicly released folders.
### How to use:
1. Make sure you have Node.js installed on your machine.
2. In your terminal, navigate to the directory with "Config.json."
3. Open "Config.json," view the `EXAMPLE_FORMAT` section for how to format the locations you would like to pull, fill in the `locations` in the same format, then save.
4. Run `node start` to initiate!
